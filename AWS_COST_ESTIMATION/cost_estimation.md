## コスト削減オプション

| 対策 | 削減額 | トレードオフ |
|------|--------|-------------|
| NAT Gateway → パブリックサブネット | **-$45/月** | セキュリティ低下（ECS が直接インターネットに露出） |
| EC2 → スケールゼロ | **-$10/月** | 30-60秒のコールドスタート |
| Reserved Instance (EC2, 1年) | **-$3/月** | 1年コミット必要 |
| ALB → 削除不可 | — | WebSocket ヘルスチェックに必要 |

---

*料金は 2026年5月時点の AWS Price List API (ap-northeast-1) に基づく。実際の請求額はデータ転送量等により変動する可能性あり。*
